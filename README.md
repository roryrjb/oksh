# oksh

__Upstream:__ https://github.com/ibara/oksh

### Distribution Support

* Ubuntu 18.04 Bionic
* Ubuntu 20.04 Focal
* Ubuntu 20.10 Groovy
* Ubuntu 21.04 Hirsute

### Installation

```
sudo add-apt-repository ppa:dysfunctionalprogramming/oksh && \
    sudo apt update && \
    sudo apt install oksh
```
